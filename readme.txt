И желательно эту папку не комитить.

для закачки модулей node_modules, через cmd:
- указываем путь ../Client
- npm install -g @angular/cli
- npm install

для запуска клиента, через cmd:
- указываем путь ../Client
- ng serve --open (само собой должен быть установлен nodejs)

для запуска тестового сервера на nodejs
- указываем путь ..nodejsServer
- node app.js
